# AI Coding模块

本模块介绍AI辅助编程方向。

## 核心技术

```
AI Coding = LLM + 代码理解 + 工具调用 + REPL

能力：
├── 代码补全（Copilot）
├── 代码生成（Codex）
├── 代码修复（Bug检测）
├── 代码审查（Code Review）
├── 测试生成
└── 文档生成
```
