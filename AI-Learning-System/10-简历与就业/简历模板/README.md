## 写作要点

### STAR法则

```
Situation: 项目背景
Task: 我的职责
Action: 具体做法
Result: 量化成果
```

### 量化技巧

```
❌ 不好：提升了系统准确率
✓ 好：准确率从65%提升到85%（提升20个百分点）

❌ 不好：优化了模型性能
✓ 好：推理速度从100ms优化到20ms（5倍加速）
```
