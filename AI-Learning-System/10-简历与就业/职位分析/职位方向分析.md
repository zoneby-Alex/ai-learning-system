# AI职位方向分析

## LLM应用工程师（最速成）

```
技能要求：LangChain + RAG + Agent + FastAPI + Docker
薪资范围：20-40K（初级），30-50K（资深）
市场需求：★★★★★ 极大
学习时间：3-6个月
代表公司：AI startup、传统企业AI部门

日常工作：
- RAG系统开发维护
- Agent工作流编排
- Prompt工程优化
- API开发和部署
```

## AI算法工程师（门槛高）

```
技能要求：PyTorch + 论文复现 + 竞赛 + 数学推导
薪资范围：30-60K
市场需求：★★★★
学习时间：6-12个月
代表公司：大厂AI Lab、明星startup

日常工作：
- 模型训练和调参
- 论文复现和改进
- 竞赛参与
- 算法落地
```

## AI Infra工程师（最高薪）

```
技能要求：CUDA + TensorRT + vLLM + 分布式 + K8s
薪资范围：40-80K
市场需求：★★★★★（极度稀缺）
学习时间：6-12个月
代表公司：推理平台、云服务商

日常工作：
- 推理性能优化
- GPU集群管理
- 模型量化部署
- 分布式训练框架
```

## 多模态工程师（最大风口）

```
技能要求：VLM + CLIP + ViT + 文档理解
薪资范围：35-60K
市场需求：★★★★★
学习时间：4-8个月

日常工作：
- VLM微调和优化
- 文档理解pipeline
- 多模态Agent
```

## Agent开发工程师（最热）

```
技能要求：LangGraph + MCP + ReAct + Multi-Agent
薪资范围：30-50K
市场需求：★★★★★
学习时间：3-6个月

日常工作：
- Agent工作流设计
- MCP工具开发
- 多Agent协作系统
```

```
快速就业 → LLM应用工程师
```
