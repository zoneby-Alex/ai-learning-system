# 职位分析模块

本模块分析AI相关岗位的市场需求和技能要求。

---

## 岗位分类

### 1. AI算法工程师

```
职责：算法研发、模型训练、效果优化

技能要求：
- 扎实的深度学习基础
- PyTorch/TensorFlow
- 论文复现能力
- 竞赛经验（加分）

薪资范围：30-60K
代表公司：大厂算法岗、AI独角兽
```

### 2. LLM应用工程师

```
职责：RAG开发、Agent编排、Prompt优化

技能要求：
- LangChain/LlamaIndex
- RAG系统开发
- FastAPI/Django
- 向量数据库

薪资范围：25-45K
代表公司：AI应用公司、传统企业AI部门
```

### 3. AI Infra工程师

```
职责：推理优化、分布式训练、模型部署

技能要求：
- CUDA/Triton
- TensorRT/vLLM
- Kubernetes
- 分布式系统

薪资范围：40-80K
代表公司：AI Infra公司、云计算
```
