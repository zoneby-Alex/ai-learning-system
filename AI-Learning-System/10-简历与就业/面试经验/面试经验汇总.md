# AI岗位面试经验汇总

## 面试流程

```
典型流程（大厂）：
1. 简历筛选
2. 在线笔试/算法题（1-2轮）
3. 技术面（2-3轮）
4. 交叉面/总监面
5. HR面
6. Offer

总时长：2-4周
```

---

## 各轮次内容

### 算法题（45分钟）

```
常见题型：
- LeetCode Hot 100
- 重点：数组/字符串/DP/二叉树/图
- 难度：Medium为主，偶尔Hard

准备：
- 每天3道，2个月后手感自然形成
- 注重代码风格和沟通
- 边写边讲思路
```

### 技术面（60分钟）

```
Part 1: 项目深挖（30分钟）
- 为什么做这个项目？
- 技术方案怎么选的？
- 遇到什么困难？怎么解决的？
- 结果如何？有什么改进空间？

Part 2: 八股文（20分钟）
- Transformer架构
- Attention计算过程
- 优化器原理
- 正则化方法
- LLM训练流程

Part 3: 手撕代码（10分钟）
- Multi-Head Attention
- ResNet Block
- 反向传播推导
```

---

## 行为面试（Behavior）

```
常见问题：
Q: 为什么离职/换方向？
A: 追求技术挑战 + AI方向前景 + 能力匹配

Q: 职业规划？
A: 短期深耕技术(1-2年) → 独立负责项目(3-5年) → 技术专家/TL

Q: 优缺点？
A: 优：学习能力强/热爱技术/善于解决问题
   缺：有时候追求完美（同时表明在改进）

Q: 期望薪资？
参考：应届20-30K，3-5年30-50K，5年+ 50K+
```

---

## 薪资谈判技巧

```
1. 不要先出价：让HR先说范围
2. 给范围不是数字：30-35K vs 30K
3. 了解市场行情：脉脉/Boss直聘/猎头
4. 考虑总包：base + 奖金 + 期权 + 福利
5. 多拿offer竞争：最有说服力
```

---

## Offer对比维度

```
- 技术成长：项目 + 团队 + 培训
- 薪资：base + 奖金 + 期权
- 公司阶段：大厂(稳定) vs 创业(风险+高回报)
- 工作强度：996 vs 965
- 团队氛围：技术氛围 + 管理风格
```
