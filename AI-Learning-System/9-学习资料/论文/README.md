## 论文阅读方法

```
1. 摘要（5分钟）→ 确定是否值得读

2. 引言（10分钟）→ 了解动机和贡献

3. 方法（20分钟）→ 核心算法

4. 实验（15分钟）→ 效果验证

5. 复现代码（2-4小时）→ 真正理解
```
