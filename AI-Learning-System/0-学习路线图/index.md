本目录包含三种学习时间规划，适合不同目标和时间投入的学习者。

## 文件列表

| 文件 | 适用场景 | 每天投入 | 总时长 |
|------|---------|---------|--------|
| [3个月速通版.md](./3个月速通版.md) | 转行/快速就业 | 6小时 | 3个月 |

## 各版本核心差异

| 阶段 | 3个月速通 | 6个月就业 | 12个月深入 |
|------|----------|----------|----------|
| 数学基础 | 压缩到2周 | 4周 | 8周 |
| 机器学习 | 跳过/略过 | 2周 | 4周 |
| 深度学习核心 | 3周 | 5周 | 8周 |
| LLM/项目 | 4周 | 6周 | 10周 |
| 论文深读 | 无 | 选读 | 精读10+篇 |
| 竞赛/开源 | 无 | 1个 | 3+个 |

## 学习优先级对照

### 3个月速通 - 只学最重要的
```
P0: Transformer / Attention / PyTorch / RAG / Agent
P1: CNN(RESNET) / LSTM / HuggingFace / LangChain
P2: ViT / Diffusion / MoE / 论文
```

### 6个月就业 - 全面覆盖
```
P0: 数学基础 / BP / Attention / Transformer / RAG / Agent
P1: CNN / RNN / LLM微调 / 项目实战
P2: ViT / Diffusion / 论文复现
```

### 12个月深入 - 大厂标准
```
P0: 数学+ML全体系 / BP / Attention / Transformer / 论文
P1: 主流模型源码 / 论文复现 / 竞赛
P2: 前沿方向探索 / 开源贡献 / 原创研究
```
