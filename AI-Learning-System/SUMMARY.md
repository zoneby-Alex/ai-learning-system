# AI学习体系 - 总结报告

## 项目概述

本项目按照"学习路径"和"就业导向"对AI/深度学习知识进行系统化整理，为有基础但知识断层的学习者提供完整的可执行学习方案。

## 项目使用指南

### 新手学习路径
1. 阅读 [README.md](../README.md) 了解整体结构
2. 选择 [学习路线图](../0-学习路线图/) 中的一个版本
3. 按模块学习每个知识点
4. 完成 [项目实战](../6-项目实战/) 中的对应项目
5. 使用 [面试题库](../8-面试题库/) 准备面试

### 项目复用方式
1. 按需查阅知识点卡片
2. 直接使用项目模板
3. 引用简历中的项目描述
4. 用作面试口述素材
