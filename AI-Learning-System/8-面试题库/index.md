## 面试评分标准

```
答案质量:
⭐⭐⭐⭐⭐ 满分：原理清晰 + 公式推导 + 代码实现 + 工程经验
⭐⭐⭐⭐   良好：原理正确 + 能推导公式
⭐⭐⭐     及格：知道是什么，但原理模糊
⭐⭐       不及格：只知道名词
```

---

## 回答技巧

### 1. 原理类问题
```
公式 → 物理意义 → 为什么这样做 → 有什么优缺点
```

### 2. 对比类问题
```
先说相同点 → 再逐条对比不同点 → 各自适用场景
```

### 3. 经验类问题
```
项目背景 → 我的角色 → 具体做法 → 最终成果
```
