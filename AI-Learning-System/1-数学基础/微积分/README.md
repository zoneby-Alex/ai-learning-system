## 核心知识点

### 1. 导数与偏导数

```python
# 导数定义
# f'(x) = lim(h->0) [f(x+h) - f(x)] / h

# 偏导数：多元函数对其中一个变量的导数
# ∂f/∂x 表示对x的偏导（y视为常数）
```

### 2. 链式法则

```python
# 一维链式法则
# d/dx f(g(x)) = f'(g(x)) * g'(x)

# 深度学习中的链式法则
# ∂L/∂W = ∂L/∂y * ∂y/∂W

# 多维链式法则
# ∂z/∂x = ∂z/∂y * ∂y/∂x
```

### 3. 梯度

```python
# 梯度是偏导数的向量
# ∇f = [∂f/∂x₁, ∂f/∂x₂, ..., ∂f/∂xₙ]

# 梯度指向函数增长最快的方向
# 负梯度指向函数下降最快的方向
```

## 面试高频问题

### Q1: 链式法则在反向传播中的作用？

**答**：
反向传播利用链式法则从后向前计算每个参数的梯度：
```python
∂L/∂W = ∂L/∂y * ∂y/∂z * ∂z/∂W
```
每一层的梯度只需要知道：
1. 上一层的梯度（来自后一层）
2. 当前层的激活函数导数
3. 当前层的输入（forward时保存）

---

### Q2: 梯度和方向导数的关系？

**答**：
- 梯度∇f是一个向量，指向函数增长最快的方向
- 方向导数D_uf = ∇f · u，表示沿方向u的导数
- 最大方向导数 = |∇f|，发生在u与∇f同向时

## 代码练习

```python
import numpy as np

# 数值导数
def numerical_derivative(f, x, eps=1e-5):
    return (f(x + eps) - f(x - eps)) / (2 * eps)

# 偏导数
def partial_derivative(f, x, i, eps=1e-5):
    x_plus = x.copy()
    x_minus = x.copy()
    x_plus[i] += eps
    x_minus[i] -= eps
    return (f(x_plus) - f(x_minus)) / (2 * eps)

# 梯度
def gradient(f, x, eps=1e-5):
    return np.array([partial_derivative(f, x, i, eps) for i in range(len(x))])
```
