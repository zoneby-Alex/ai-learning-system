## 快速开始

```bash
npm install          # 安装依赖
npm run docs:dev     # 启动开发服务器 → http://localhost:5173
npm run docs:build   # 构建生产版本
```

## 技术栈

- **框架**：[VitePress](https://vitepress.dev/)（Vue 官方文档工具）
- **部署**：GitHub Pages + GitHub Actions 自动构建
- **搜索**：本地全文搜索（无需外部服务）
- **数学**：MathJax 渲染
